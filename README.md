# Лабораторная работа №5 — Agile и Scrum

Моделирование Scrum-процесса с использованием GitHub Issues.

Это веб-приложение для управления задачами пользователей (ToDo-лист).  
Пользователи могут регистрироваться, входить в систему, создавать, просматривать, редактировать и удалять свои задачи.  
Каждая задача имеет название, описание, статус и дату создания.  
Приложение реализовано на Python с простой доменной моделью.  
В рамках лабораторной работы смоделирован один спринт: создан Product Backlog, выбран спринт, оформлены задачи с labels, приоритетами и оценками сложности, смоделирован ход спринта и написаны Sprint Review и Retrospective.
