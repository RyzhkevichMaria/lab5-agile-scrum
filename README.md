## Sprint Retrospective

Что прошло хорошо:
- Удалось приоритизировать и завершить ключевые задачи аутентификации.
- Хорошее планирование — выбраны самые важные функции для первого спринта.

Что пошло не так:
- Некоторые задачи оказались сложнее, чем ожидалось, и не были завершены.

Что можно улучшить в следующем спринте:
- Лучше оценивать сложность задач (story points).
- Добавлять больше времени на тестирование и доработку.
- Раньше начинать работу над задачами с высокой оценкой.
